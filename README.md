# EstudioLab - CRM & Panel Administrativo Jurídico

Sistema de gestión de clientes, correspondencia de Gmail y seguimiento de trámites para el estudio jurídico **EstudioLab**.

## Características Principales

- 👥 **Gestión de Contactos y Clientes**: Clasificación inteligente entre contactos generales y clientes activos/inactivos.
- 📋 **Fichas de Clientes 360°**: Visualización centralizada de información personal, estado de trámites e historial completo.
- ✉️ **Integración con Gmail**: Registro automático de correos enviados y recibidos asociados a cada cliente.
- 🎙️ **Redacción por Voz con IA (N8N)**: Dictado directo de correos con micrófono y transcripción automática mediante N8N y OpenAI Whisper.
- ✨ **Formato Bonito IA**: Estructuración y pulido profesional de mensajes en un solo clic.
- 📊 **Sincronización con Google Sheets**: Mantiene actualizada automáticamente la pestaña `fichas_clientes` en tu planilla de Google.

## Comenzar

Ejecuta el servidor de desarrollo local:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
